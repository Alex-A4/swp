Предварительные приготовления:
Все шрифты находяться в репозитории  https://git.sbis.ru/sbis/controls/.
Если у вас нету этого репозитория, его необходимо выкачать к себе. О том, как выкачивать
репозиторий, подробно описано здесь
https://wi.sbis.ru/doc/platform/workplace-settings/windows/#tortoise-settings
Если не установлен Git, то действуйте пошагово по инструкции
https://wi.sbis.ru/doc/platform/workplace-settings/windows/#git
Если у вас нет прав developer для репозитория, тогда вам нужно написать поручение
в отдел по дежурству сборки с просьбой предоставить доступ к репозиторию. Пример такого поручения
https://online.sbis.ru/opendoc.html?guid=897aa82f-0923-4a59-9ab2-5c4bd2dc9f7a
После всех вышеописанных приготовлений перейдите в директорию контролов.
Шрифты к cbuc-icons расположены в директории
для 16 : components/themes/fonts/cbuc-icons
для 24 : components/themes/fonts/cbuc-icons24
В контролах добрасываются только 3 расширения шрифтов: .eot, .woff и .woff2.
Первые 2 можно найти в папке шрифта от проектировщиков, например
T:\Документы отделов\Отдел оформления\СБиС\Общее\Прототипирование\Шрифт\ver\1_12_93\24\fonts
.woff2 генерируется вручную по инструкции ниже.

Инструкция по доброске шрифтов(если нужно просто добросить шрифт, новых иконок нет):

1. Используя онлайн-конвертер https://transfonter.org/, сконвертируйте шрифты из .woff в .woff2

	1.1 Отключить всё кроме Family support, галочка должна стоять только на WOFF2
	
	1.2 Загрузить файл со шрифтами в разрешение .woff и нажать Convert
	
	1.3 После завершения конвертации сайтом, скачать полученный архив, нажав Download.
	Извлечь файл со шрифтом с расширением .woff2 . Для удобства можете положить его в папку с новыми
	.eot и .woff.

   !!! Из скачанного архива вам необходим только файл с расширением .woff2 !!!

2. Из директорий
components/themes/fonts/cbuc-icons
и
components/themes/fonts/cbuc-icons24
удаляем старые шрифты. На их место кладём новые.
!!! Внимание, кладём только 3 файла - .eot, .woff и .woff2. Также не перепутайте размеры иконок.
у 24 размера файлы имеют название cbuc-icons24.woff2, у 16го cbuc-icons.woff2 !!!

3. В результате у вас должно получиться 6 изменённых файлов для доброски:
3 из директории components/themes/fonts/cbuc-icons
3 из директории components/themes/fonts/cbuc-icons24

4.(если есть новые иконки, этот пункт пропускайте) Коммитим и пушим. Оформляем доброску на онлайне.

Инструкция по доброске шрифтов(есть новые иконки):

Все иконки описаны в components/themes/fonts/glyphs.less, все новые иконки докидываются туда.
Проектировщики прикладывают к задаче ссылку до нового шрифта в виде
T:\Документы отделов\Отдел оформления\СБиС\Общее\Прототипирование\Шрифт\ver\1_12_93

1. Прежде всего убедитесь, что для 24 и 16 размеров иконок content одинаковый. Стили с новыми иконками можно найти:
для 16 <версия шрифта>/16/style.css
для 24 <версия шрифта>/24/style24.css
Вам остаётся проверить, что в обоих css-файлах для новых иконок content совпадает. Если да, тогда можете конвертить в woff2
по инструкции выше. Если нет, пишете об этом автору задачи, он поправит шрифт и пришлёт ссылку на новую версию. Повторяем проверку
до получения одинакового content.
2. Выполните все действия из инструкции по доброске непосредственно шрифта.
3. В файл components/themes/fonts/glyphs.less добавляем новые иконки.
4. Готово, оформляем доброску. В доброске у вас должны быть следующие изменения:
components/themes/fonts/cbuc-icons/ - должно быть 3 изменённых файла - eot, woff и woff2
components/themes/fonts/cbuc-icons24/ - должно быть 3 изменённых файла - eot, woff и woff2
components/themes/fonts/glyphs.less - должны быть добавлены новые иконки(опционально, если необходимо добавить новые иконки).
Пример такой доброски можете посмотреть вот здесь:
https://online.sbis.ru/opendoc.html?guid=12322a38-29ec-449c-ad46-a39affecd8b4
